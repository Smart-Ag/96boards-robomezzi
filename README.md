# 96boards-robomezzi

This repository contains the hardware design files for a robotics mezzanine
board that plugs into 96boards CE based products.

## Hardware Specs

- 16 PWM servo outputs
- 8 5V GPIO
- 4 5V Analog Inputs
- 9DOF sensor
- 6V servo switching regulator
- 5V I2C header
- FTDI debug UART
